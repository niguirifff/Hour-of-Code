#Hour of Code

**Objetivo:** Integrar efectos de sonido y música en el proyecto Hour of Code de Epic utilizando el middleware de audio Wwise. Este trabajo se enfoca exclusivamente en la implementación de audio y música. No es necesario crear ningún recurso (asset), ya que todos los sonidos serán proporcionados por el docente.

**Consigna:** Los estudiantes recibirán un proyecto en Unreal Engine a fin de llevar a cabo la implementación integral de audio del proyecto utilizando el middleware Wwise.

**Materiales:**

- Proyecto de Unreal Engine 5.5.4 Hour of Code de Epic Store.
- Carpeta con los archivos de audio.

**Lo fundamental para aprobar esta actividad es la implementación.** No se evaluará ni se tomará en cuenta la composición ni la adición de música y efectos de sonido adicionales; estos elementos no afectarán la calificación.

**Pasos para la realización del trabajo:**

1.  Clonar el repositorio.
2.  Integrar Wwise en el proyecto de Unreal Engine utilizando el Audiokinetic/Wwise Launcher.
3.  Crear eventos y llevar a cabo la implementación de los archivos de audio en el proyecto.
4.  Una vez que todos los sonidos estén implementados, comprimir el proyecto de Unreal y subirlo a Moodle.

**Versiones de software necesarias:**

- Unreal Engine 5.5.4 - Se instala de la Epic Store de EPIC GAMES.
- Audiokinetic Wwise 2024.1.4.8780.3614 - Se instala desde el Wwise/Audiokinetic Launcher.

**Sonidos a implementar en la actividad:**

1.  Music
2.  Ambience
3.  CharacterFoley
4.  CharacterFoley_Jump-Land
5.  CharacterFootsteps
6.  CharacterJumpVO
7.  Checkpoint
8.  CoinPickUp
9.  Other (Platform Hum, Shrads Hum, Door, Power Up)

Esta actividad se subirá directamente a Moodle, sin excepción.
